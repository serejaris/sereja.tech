## Настройка

Весь процесс — от пустого сервера до работающего агента в Telegram — занял один стрим. Вот что происходило по порядку.

Сначала — SSH. Contabo выдаёт IP и пароль. Я подключился через Warp — терминал, который умеет работать с SSH-сессиями удобнее обычного. Но это не принципиально, подойдёт любой терминал.

Заранее сгенерировал SSH-ключ и закинул на сервер через `ssh-copy-id`. Чтобы не вводить пароль каждый раз — особенно когда стримишь и не хочешь светить его на камеру.

OpenClaw ставится одной командой — `curl` с их сайта. Агент развернулся, попросил API-ключ. Вставил ключ от Kimi.

Дальше — Telegram. Завёл бота через BotFather, скопировал токен, скинул в OpenClaw. Всё.

```
[Contabo VPS]
     │
     ├── OpenClaw (агент)
     │      ├── Kimi K2.5 API (мозг)
     │      └── Telegram Bot API (интерфейс)
     │
     └── [Телефон] → Telegram → Бот → OpenClaw → Kimi → ответ
```

Без Docker, без оркестрации, без CI/CD. Два токена — и агент живой.

## Безопасность

Исследователи нашли [42 665 открытых инстансов OpenClaw](https://clawctl.com) в интернете. Сорок две тысячи серверов без защиты. Любой может зайти, прочитать переписку, украсть ключи.

Я уже разбирал эту проблему [в посте про Clawdbot](/blog/clawdbot-personal-ai-assistant). Цифры выросли с тысячи до 42 тысяч. Проблема масштабировалась вместе с популярностью.

Что стоит сделать:

- **Firewall.** Закрыть все порты, кроме SSH и тех, что нужны OpenClaw
- **Отдельный сервер.** Не ставить на машину с важными данными
- **Ограничить права.** Агент не должен работать от root
- **Менять ключи.** API-ключ — как пароль. Ротация раз в месяц минимум
- **Мониторинг.** Логи, алерты, аудит команд

42 тысячи человек не сделали эти пять шагов. Не повторяй.

## FAQ

### OpenClaw — это тот же Clawdbot?

Да. Проект переименовывался дважды: Clawdbot → Moltbot → OpenClaw. Кодовая база та же, создатель тот же — Peter Steinberger. Сменилось только название.

### Почему Kimi, а не Claude или GPT?

Цена. Входные токены Kimi K2.5 стоят $0.60 за миллион — в 8 раз дешевле Claude Opus. Для агента, который работает круглосуточно и обрабатывает рутинные задачи, это критичная разница. Для сложных задач, где нужно глубокое рассуждение, Claude всё ещё лучше.

### Безопасно ли ставить OpenClaw на VPS?

Если настроить правильно — да. Firewall, непривилегированный пользователь, закрытые порты, ротация ключей. Если поставить «из коробки» и забыть — нет. 42 тысячи открытых инстансов это подтверждают.

### Сколько токенов тратит агент в день?

Зависит от нагрузки. На полную настройку — системный промпт, характер, лендинг — ушёл 1% недельного лимита Kimi Coding Plan. При повседневном использовании (десяток сообщений в день) $1 в месяц на API — реальная цифра.
