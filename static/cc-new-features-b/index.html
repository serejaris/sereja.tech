Что произошло
Исходный код Claude Code — CLI-инструмента Anthropic для разработчиков — попал в открытый доступ. npm-пакет @anthropic-ai/claude-code разобрали целиком: 1 884 файла, около 33 мегабайт кода.
Внутри обнаружились десятки функций, которых нет в документации: виртуальный питомец, долговременная память со «сновидениями», фоновые агенты, удалённое управление, голосовой ввод — и это далеко не всё. Часть возможностей полностью реализована, но отключена флагами. Другие управляются с серверов Anthropic и могут быть включены для любого пользователя в любой момент.
